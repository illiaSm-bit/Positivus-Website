## How to Run the Project
1. Download the project files
-Clone the repository or download the ZIP file.
2. Open the project in a browser
-Navigate to the project folder.
-Open the index.html file with any modern web browser (Chrome, Firefox, Edge, Safari).
3. Enable real-time CSS updates with Sass (optional)
-Make sure you have Sass installed.
-Run the following command in your terminal to watch your SCSS file and automatically compile it to CSS whenever you make changes:
 sass --watch ./style/index.scss:style.css
4. Optional:
-For a better development experience, you can use a local server (e.g., VS Code Live Server extension) to enable hot reloads.
